What is Identifier

  • Identifier is a specific name in Java program .
  • It can be class name, interface name, method name, variable name .
  • In below program all red circles are Identifiers .




Rules For Identifier

  • Java Identifier allowed a-z, A-Z, 0-9, _ and $ characters .
  • Java Identifier should not start with digits .
  • Java Identifier are case sensitive .
  • We can not use reserve keywords for java identifier .
If we will not follow the rules for Identifier we will get compile time error . We can use class name and interface name as java identifier, but it is not a good programming practice .
