BufferedReader

  • We can use BufferedReader to read character data from the file .
  • The main advantage of BufferedReader over FileReader is we can read data line by line instead of character by character which is more convenient to the programmer .
Constructors :
  • BufferedReader br = new BufferedReader(Reader r);
  • BufferedReader br = new BufferedReader(Reader r, int buffersize);
  • Note: BufferedReader can not communicate with a file, it can communicate by a some Reader object .
Methods :
  • int read()
  • int read(char[] ch)
  • void close()
  • String readLine()
  • It attempts to read next line from the file and returns it . If the next line is not available then we will get null .
  • Note: Whenever are closing BufferedReader automatically underlying Readers will be closed and we are not required to close explicitly .
