FOR

  • This is the most commonly used loop in java .
  • For loop is the best choice if we know the no of iterations in advance .
  • Curly braces are optional and without curly braces we can take only one statement, which should not be declarative statement .

Initialization Section :

  • This part will be executed only once .
  • Usually we can declare and initialize local variable of for loop in this section .
  • Here we can declare any no. of variable but should be of the same type . i.e we can not declare multiple variable of different data types .
  • In initialization section, we can take any valid java statement including "System.out.println" also .

Conditional Expression :

  • Here we can take any valid java expression, but should be boolean type .
  • This conditional expression part is optional and if we are not taking anything then compiler will always take it as "true"

Increment/Decrement Section :

  • Here we can take any valid java statement including "System.out.println" statement also .
  • All the three parts of for loop is independent of each other and optional .
