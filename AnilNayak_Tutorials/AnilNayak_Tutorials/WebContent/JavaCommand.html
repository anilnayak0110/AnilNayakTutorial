Java Commands :


javac command:
  • We can use javac command to compile a single or group of java source files .
  • javac compile the java file like Test.java
  • After compiling java file it creates dot(.) class files, which contains byte code of java class .
  • If a java file contains more than one class, then the java compiler can creates those number of dot(.) class file .
java
  • We can use java command to run a single class .
  • java command execute the class name of java file, which contains the java main(String[] args) method .
  • Without compilation of java file we can not execute java command directly , because it except dot(.) class file to execute .
  • By using java command we can pass command line arguments also
  • java execution happen by Java Virtual Machine(JVM)
  • Note: We can compile any number of class at a time, but we can execute only one class at a time .
