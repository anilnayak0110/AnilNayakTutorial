WeakHashMap
  • It is exactly same as HashMap except the following difference
  • In case of Normal HashMap , if an object associated with HashMap, then it is not eligible for Garbage Collection, eventhough it does not contain any references i.e , HashMap dominates Garbage Collector .
  • But in case of WeakHashMap , if an object does not contain any references then it always eligible for Garbage Collector eventhough it is associated with WeakHashMap i.e Garbage Collector dominates WeakHashMap .
