Vector:

  • The underlying data structure is Resizable array or Growable array .
  • Insertion order is preserved .
  • Duplicate Objects are allowed .
  • Heterogeneous objects are allowed .
  • Null insertion is possible .
  • Implements serializable, cloneable and RandomAccess interfaces .
  • Every method present inside Vector is synchronized and hence Vector object is Thread safe .
Constructors:
  • Vector v = new Vector();
    Creates an empty Vector object with default initial capacity 10 .
  • Once Vector reaches its maximum capacity then a new Vector object will be created with
    new capacity = 2*current capacity;
  • Vector v = new Vector(int initialcapacity)
  • Vector v = new Vector(int initialcapacity, int incrementalcapacity);
  • Vector v = new Vector(Collection c);
