IS-A-Relationship:

  • It is also known as Inheritance .
  • By using extends keyword we cab implement IS-A-Relationship .
  • By using IS-A-Relationship we can achieve re-usability of the code .
  • Whatever the Parent class has by default available to child class . Hence on the child class object we can call both parent class and child class method .
  • Whatever has child class by default not available to child class . Hence by using parent class object we can't access child class method
  • Parent class reference can be used to hold child class object . But by using parent class reference we can call only the methods of parent class and we can not call child specific methods .
  • We can not use child class reference to hold parent class object .
  • Entire Java API is implemented based on the Inheritance concept .
  • Whatever the common methods which are applicable for any Exception or Error are defined in Throwable class . Hence Throwable class acts as the root for java exception hierarchy .
  • Whatever the common methods which are applicable for any java object are defined in Object class . Hence Object class acts as a root for all java classes .
When we will use Inheritance ? :
  • When we need all the data member and methods from the parent class then we should go for inheritance . Otherwise we should not go for inheritance to avoid performance problem .
Why multiple inheritance is not supported by java ? :
  • There may be a chance of ambiguity problem and hence multiple inheritance is not allowed in java .
  • But an interface can extend more than one interface at a time and hence multiple inheritance is allowed only for interface .
  • An interface can extends multiple interface at a time .
  • In the interfaces eventhough multiple method declaration are there but implementation is unique . Hence there is no chance of ambiguity problems in interface .
  • If our class does not extends from any other class class, then only our class is the direct child class of Object .
  • If our class extends any other class then it is not direct child class of Object .
