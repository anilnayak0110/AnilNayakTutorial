Data types

In Java to declare variable we need a type is called data type .
  • Primitive Data Types
    • Numeric data types
      • Integral data types
        • byte, short, int, long
      • Floating point data types
        • float, double
    • Non-numeric data types
      • boolean, char
Except boolean and char remaining data types are called signed data type, because they have both +ve and -ve numbers .
byte:
  • Size: 1 byte(8 bits)
  • MAX_VALUE: +127
  • MIN_VALUE: -128
  • Range: -128 to +127
  • Default value: 0
byte data type mostly we used to handle data in terms of streams, in terms of file or in terms of network .
short:
  • Size: 2 byte(16 bits)
  • MAX_VALUE: +32767
  • MIN_VALUE: -32768
  • Range: -32768 to +32767
  • Default value: 0
int:
  • Size: 4 byte(32 bits)
  • MAX_VALUE: +2147483647
  • MIN_VALUE: -214748368
  • Range: -231 to +231-1
  • Default value: 0
int data type is mostly common used data type in java .
long:
  • Size: 8 byte(64 bits)
  • MAX_VALUE: +263
  • MIN_VALUE: -263-1
  • Range: -263 to +263-1
  • Default value: 0l
long data type is used sometimes to hold big values in java .
float:
  • Size: 4 byte(32 bits)
  • MAX_VALUE: +3.4e38
  • MIN_VALUE: -3.4e38
  • Range: -3.4e38 to +3.4e38
  • Default value: 0.0f
float data type is used when we want 5 to 6 decimal places of accuracy .
double:
  • Size: 8 byte(64 bits)
  • MAX_VALUE: +1.7e308
  • MIN_VALUE: -1.7e308
  • Range: -1.7e308 to +1.7e308
  • Default value: 0.0d
double data type is used when we want 14 to 15 decimal places of accuracy .
char:
  • Size: 2 byte(16 bits)
  • MAX_VALUE:65535
  • MIN_VALUE:0
  • Range: 0 to 65535
  • Default value: space character
boolean:
  • Only allowed value: true and false
  • Default value: false
The default value for object reference is null .
