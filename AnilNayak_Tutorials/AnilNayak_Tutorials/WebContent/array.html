Array

Introduction
  • Array is an index based collections of homogeneous data elements .
  • When we wants to assign multiple values to a single variable, then we should go for array .
  • The problem in array is it is fixed in size, once we declare an array with some size we can not increase or decrease size of the array, so before defining an array we should know the size.

Array Creation
  • Every array in java is an object, hence we can create an array by using new operator .
    int[] a = new int[6];
  • For every array type there is corresponding classes are available .
    Array TypeClasses
    int[][I
    int[][][[I
    byte[][B
    short[][S
    long[][J
    float[][F
    double[][D
    boolean[][Z
    char[][C
  • At the time of array creation compulsory we should give the size, else we will get Compile time error .
    int[] x= new int[]Compile Time error
  • We can create an array with zero size
    int[] x = new int[0];Correct
  • If we give the array size with a negative number, then we will get Runtime Exception:NegativeArraySizeException
    int[] x = new int[-7];NegativeArraySizeException
  • To specify array size the allowed data types are :
    byte
    short
    char
    int
  • The maximum allowed array size in java is 2147483647
Array Initialization
  • Once we created an array, then every array element is by default initialized with default value .
  • If we are trying to access array element with out of range then we will getArrayIndexOutOfBoundsException

  • length is the final variable applicable for arrays .

Anonymous Arrays
  • Sometimes we can declare an array without any name is called anonymous array.
  • The purpose of anonymous array is instant use .
    new int[]{20,30,40};
  • While creating anonymous array we can not specify size, or else we will get compile time error .
Array element assignments:
  • For primitive type arrays as array elements we can provide any type which can be implicitly promoted to declared type .
  • For int type arrays, the allowed element types are byte,short,char and int .
  • For float type arrays, the allowed element types are byte,short,char,int,long and float .
  • For object type arrays as array elements we can provide either declared type objects ar its child class objects .
  • Whenever we are assigning one array to another array dimensions and types should be matched .
