Why two JRE in Java

When we install java, check the installation locations as I shown below screen shots





As you can see there is two jre, one is outside jdk and another is inside jdk. So why is it like that ,actually to run any compiled java class we need only JRE . But to both for developing java class, compiling and executing we need JDK and JRE . So when any non-developer people only wants to run only compiled java class, then outside jdk 'JRE' is sufficient.But wnen we wants both development and executions of java class, then we need both JDK and JRE .
