Bitwise Operators(&,|,^)

  • &(AND) : If both arguments are true then only result is true .
  • |(OR) : If at least one argument is true then only result is true .
  • ^(X-OR) : If both are different arguments then only result is true .
System.out.println(true&false)//output : false
System.out.println(true|false)//output : true
System.out.println(true^false)//output : true
System.out.println(true&true)//output :true

Bitwise complement operator(~)

  • We can apply this operator only for integral types but not for boolean types .
System.out.println(~true)//output : C.E: operator ~ can not be applied to boolean

Boolean complement operator(!)

  • We can apply these operator only for boolean type, not for integral type .
System.out.println(!4)//output : C.E: operator ~ can not be applied to int
System.out.println(!true) //output : false
System.out.println(!false) //output : true
