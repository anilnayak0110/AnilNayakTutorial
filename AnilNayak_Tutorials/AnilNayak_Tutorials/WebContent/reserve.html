Reserved words

In java some words are reserve to do some functionality is called reserved words .
  • Reserved words
    • Key words
      • Unsed Keywords
        • goto,const
      • Used Keywords
        • Keywords for datatypes
          • byte, short, int, long, float, double, char, boolean
        • Keywords for flow control
          • if, else, switch, case, default, do, while, for, break, continue, return
        • Keywords for modifiers
          • public, private, protected, default, final, static, abstract, native, synchronized, volatile, transient
        • Keywords for Exception Handling
          • try, catch, finally, throw, throws
        • Keywords for classes and interfaces
          • class, interface, package, import, extends, implements
        • Keywords for objects
          • new, super, this, instanceOf
        • keywords for no return type
          • void
        • keywords for represent group of constants
          • enum(when we wants to represent a group of constants then we should go for enum .)
          • e.g: enum SoftwareCompany { EIDIKO, IBM, TCS; }
    • Reserve Literals
      • true,false(Values for boolean data type), null(default value for object reference)
We can not use unused keywords like goto and const in java, if we will try then we will get compile time error, instead of const in java we can use final keyword .
