Equality Operators(==,!=)

  • We can apply relational operators for every primitive type including boolean type .
System.out.println(10==20)//output : false
System.out.println('a'==b)//output : false
System.out.println('a'==97)//output : true
System.out.println(true==true)//output :true.
    • we can apply relational operators for object types also.
      For object references a1 and a2 .
      a1 == a2 true if and only if both a1 and a2 pointing to the same object.
      i.e == operator used for reference comparision or address comparision .
    • To use the equality operators between object type compulsory these should be some relation between argument types(child to parent,parent to child), Otherwise we will get Compiletime error incompatible types .
    • For any object reference of on a == null is always false, but null==null is always true .
