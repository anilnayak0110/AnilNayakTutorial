Assignment Operator

  • There are three types of Assignment Operators
    • Simple Assignment Operator
    • Chained Assignment Operator
    • Compound Assignment Operator
Simple Assignment Operator
int x = 10;
Chained Assignment Operator
int a,b,c,d;
a=b=c=d=10
System.out.println(a+" "+b+" "+c+" "+d);
output : 10 10 10 10
  • We can not use chained assignment at the time of declaration Ex: int a=b=c=d=10; C.E : Can not find symbol b
