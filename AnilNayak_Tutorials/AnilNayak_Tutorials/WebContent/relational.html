Relational Operators(<,<=,>,>=)

  • We can apply relational operators for every primitive type except boolean type .
System.out.println(10>10.5)//output : false
System.out.println('a'>95.5)//output : true
System.out.println('z'>'a')//output : true
System.out.println(true>false)//output : C.E : can not be applied to boolean.
    • we can not apply relational operators for object types .
    System.out.println("Anil">"Heera")//output : C.E: can not be applied to java.lang.String
      • Nesting of relational operator is not allowed .
      System.out.println(10<20<30)//output : C.E: can not be applied to boolean,int
