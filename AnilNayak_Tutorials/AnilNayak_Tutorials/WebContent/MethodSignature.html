MethodSignature:

  • In java, method signature consists of name of the method followed by argument types .
  • Return type is not part of the method signature .
  • Compiler will use method signature while resolving method calls .
  • Within a class two methods with the same signature not allowed, otherwise we will get compile time error .
Note : public static void m1(int i, float f) Method Signature : m1(int,float)

Compile Time Error : m1(int) is already defined in Test .
