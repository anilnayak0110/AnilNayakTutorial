Encapsulation:

  • The process of binding data members and corresponding methods into a single module is called encapsulation .
  • If any java class follows data hiding and abstraction, such type of class is also to be encapsulated class .
  • The main advantages of Encapsulation are :
    • We can achieve security .
    • Enhancement will become easy .
    • It improves modularity and maintainability of the application .

Tightly Encapsulated class:

  • A java class is said to be tightly encapsulated class if all its data members declared as private .
  • Note : If the parent class is not tightly encapsulated then no child class is tightly encapsulated .
