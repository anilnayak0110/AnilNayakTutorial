Conditional Operator(?:)

  • The only ternary operator in java is conditional operator
  • We can also do nesting of conditional operator in java
Difference between new vs newInstance():
  • We can use new operator to create object if we know the class name at the beginning .
  • newInstance() is a method present in "Class" class which can be used to create object
    if we don't know the class name at the beginning and it is available dynamically at runtime .
Difference between ClassNotFoundException and NoClassDefFoundError:
  • If hard coded class name is not available at runtime then we will get runtime exception :NoClassDefFoundError, which is unchecked Exception
    Example: Test t = new Test();
    At runtime, if "Test.class" file is not available then we will get runtime Exception saying: NoClassDefFoundError
  • When dynamically provided class is not available at runtime then we will get runtime exception: ClassNotFoundException
    Example: Object o = Class.forName("Test").newInstance();
    If "Test.class" is not available at runtime then we will get runtime exception saying ClassNotFoundException
